## Api-reference
Consulta la sección de API Reference o el endpoint específico de [Login](/api-reference/authentication/login) para más detalles sobre cómo obtener tu token. (*Nota: Ajusta el enlace `/api-reference/authentication/login` a la ruta real donde documentarás el endpoint de login en Mintlify*).

> **Nota importante**: Esta API está destinada únicamente para uso autorizado.
> El acceso no autorizado está prohibido y será monitoreado.

## Servidor de Desarrollo

La API está disponible para desarrollo en la siguiente URL base:

`http://localhost:52000`
