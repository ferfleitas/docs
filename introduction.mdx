## Documentación de la API - Pedidos de Botellones

Esta API permite la gestión de clientes, pedidos y entregas de botellones de agua.

## API Reference
Consulta la sección de [API Reference](/api-reference) para ver detalles de cada endpoint disponible, parámetros requeridos y respuestas esperadas.

> **Nota importante**: Esta API está pensada para ser usada por operadores autorizados del sistema de pedidos. El acceso indebido será restringido y monitoreado.
