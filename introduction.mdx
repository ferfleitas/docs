La **BANCAME API** permite la integración de la pasarela de pagos en sitios web externos (como WooCommerce, entre otros), la generación de iframes de pago y el seguimiento de transacciones desde el portal de comercio de Bancame.

## API Reference
Consulta la sección de [API Reference](/api-reference) para ver detalles de cada endpoint disponible, parámetros requeridos y respuestas esperadas.

> **Nota importante**: Esta API está destinada exclusivamente para comercios y operadores autorizados. Todo acceso indebido será monitoreado y bloqueado.
