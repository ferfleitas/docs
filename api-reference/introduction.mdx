## Introducción

Este backend está diseñado para integrar pasarelas de pagos y gestionar el manejo de transacciones, incluyendo funcionalidades para generar iFrames de pago, procesar cobros en plataformas externas (como WooCommerce), consultar el estado de las operaciones y realizar un seguimiento detallado de cada transacción.  

Está construido con **Node.js**, implementando un enfoque **REST API** con **Express** como framework principal y utilizando **PostgreSQL** como sistema de base de datos.
