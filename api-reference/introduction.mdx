## Introducción

Este backend está diseñado para gestionar pedidos de botellones de agua, incluyendo funcionalidades para registrar clientes, crear pedidos, actualizar su estado, y generar reportes básicos. 

Está construido con **Node.js**, utilizando **Express** como framework principal y **PostgreSQL** como sistema de base de datos.

## Funcionalidades Principales
